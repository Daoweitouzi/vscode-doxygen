# vscode-doxygen
Doxygen language support for VS Code

### Installing the extension Locally

Just clone this repository under your local extensions folder:
* Windows: `%USERPROFILE%\.vscode\extensions`
* Mac/Linux: `$HOME/.vscode/extensions`
